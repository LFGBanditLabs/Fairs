# Contracts (Hardhat)

Scaffold for Hardhat contracts.

Quick start:

1. cd contracts
2. npm install
3. npx hardhat node   # optional local node
4. npx hardhat compile
5. npx hardhat test

To deploy to the local node started with `npx hardhat node` run:

1. In one terminal: npx hardhat node
2. In another terminal: node scripts/deploy.js
